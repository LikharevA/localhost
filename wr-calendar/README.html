:: Назначение скрипта ::
Скрипт показывает календарик на текущий месяц, подсвечивая текущую дату. Единственная причина, побудившая меня написать такой скрипт - отсутствие в инете простых календарей в вертикальном формате.
Календарик компактен - занимает совсем немного места - 2 048 байт :-)
:: Возможности ::
    - имеется возможность легко изменить цвет подсветки сегодняшней даты - переменная $hcolor="#884422";
    - можно изменить цвет бордюра таблицы - переменная $bcolor="#cccccc";
    - есть возможность включить/отключить печать шапки со стилями (если вы хотите использовать свои настройки на странице - то присвойте "0" этой переменной): $printshapka="0";
:: Установка и использование ::
1. Закачайте в любую папку вашего сайта файлы calendar.php и calendarjs.php Выставите им атрибуты Chmod 755.
2. Попытайтесь запустить скрипт ссылкой такого типа: http://Мой_сайт/папка_где_лежит_скрипт_на_сервере/calendar.php
Если Вы видите календарь на месяц - значит скрипт успешно работает и можно переходить к использованию. 3. Скрипт calendarjs.php можно запускать с html-странички. Для этого необходимо вставить на Вашу html-страницу следующий код: <script language="JavaScript" src="calendarjs.php">
4. При проблемах с запуском скрипта убедитесь, что Вы установили права доступа чтения/выполнения для папки в которой расположен скрипт. Если скрипт не заработал, не бойтесь экспериментировать: изменяйте атрибуты файла. На "сомнительном" хостинге, скрипт может запускаться с атрибутами 777 (вместо 755).
5. На Вашей php-страничке добавьте строчку с кодом: include "calendar.php"; в том месте, где Вам нужен календарь. И будет Вам счастье ;-) А вот на html-страничку инклюдить этот скрипт нельзя.
P.S. Внимание! Скрипт написан на PHP и не работает на бесплатном хостинге НЕ поддерживающего эту язык, например, www.narod.ru.
Если все пункты вы выполнили, а скрипт так и не запустился, ну чтож, заходите на форум по адресу: http://wr.kovostok.ru/wr-forum/index.php и в разделе "WR-Scripts" просмотрите темы, возможно решение Вашей проблемы уже есть на форуме. Если нет - задавайте вопросы. Постараюсь Вам помочь.
